* Variables: Espacios de memoria dedicados a guardar información, la cual se podrá usar mas tarde.

* Stings: Tipo de dato.
          Son bloques de texto que van siempre entre comillas, ya sean dobles ('') o simples ('). 
          Pueden ser oraciones,letras,numeros, puntuación, etc.
          Basicamente cualquier cosa que vaya entre comillas.

* Funciones: Bloque de código diseñado para realizar una tarea en particular.
             La caracteriztica más importante de las funciones es que se pueden invocar(llamar).
- Argumentos: Son los valores que recibe una función. 
- Return: Esta instrucción se utiliza para terminar con la ejecución de una función o bloque de código.
          Puede devolver un valor.       

*Declaraciones: Es como se llama a las instrucciones en JS.
- if: es una estructura utilizada para tomar decisiones.

* Valores Booleanos (Booleans): Tipo de dato.
                                Tiene dos valores : - verdadero
                                                    - falso.
